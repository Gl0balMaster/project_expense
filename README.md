Project\_expense

проект создан для подсчета своих расходов, расчета их по дням/месяцам/количеству
===

# 

# в данном проекте можно регистрироваться/логиниться и добавлять/удалять/редактировать и помечать свои расходы 

# 

# так же реализована страница для администрирования

# 

# для локального запуска необходимо:

# склонировать репозиторий: 

# git clone https://github.com/Gl0balMaster/project\_expense

# 

# установить необходимые библиотеки:

# pip install -r requirements.txt

# 

# перейти в папку и создать миграции: 

# python.exe manage.py makemigrations 



# активировать миграцию:

# python.exe manage.py migrate 

# 

# запустить сервер:

# python.exe manage.py runserver

# &nbsp;

# перейти по ссылке: 127.0.0.1:8000

# 

# (опционально) для создания суперпользователя:

# python.exe manage.py createsuperuser

# 

# для просмотра страницы суперпользователя перейти по адресу 127.0.0.1:8000/admin
